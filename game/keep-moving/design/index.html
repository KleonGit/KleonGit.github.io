Keep Moving策划

发布于 二月 26, 2021    更新于 五月 21, 2021

Keep Moving是一款高度Rogue Like化的外星主题的2D俯视角射击游戏

背景设定

玩家作为异星探险家，飞船解体坠毁，需要收集材料和零件重建飞船逃出生天。

核心特色

  • RogLike化的敌人生成，包括外观，行为，技能
  • 游戏时间相关的房间和敌人生成
  • 自由组合的武器系统

核心玩法

目标

  • 使用关键零件和材料组成飞船，完成游戏

地图

  • 地图可划分为基本的房间，由WxH个房间构成，W与H取决于星球属性，初始建议W(横)H(纵)为12x7
  • 在概念上所有房间共同拼成星球，左右/上下回绕，房间形状可以不同，位置可以小幅移动
    • 左右直接回绕
    • 上下并不直接回绕，极点附近向上/下与当前纬度+星球总纬度/2的房间相连
  • 地图横向为经度，纵向为纬度
    • 温度/亮度随经度循环变化，左移或右移取决于星球属性（自转方向）
    • 移动时间以进入房间次数（N）为准，N取决于星球属性
    • 温度变化率随纬度变化，从赤道到极点不断减弱
  • 房间分为普通房间和专用房间，专用房间可由多个房间拼接而成。
  • 地图形状未探索时不可知，探索后计入地图
  • 地表障碍物为tile-based
  • 专用房间包括：
    • 起始房间，飞船坠毁点，需要回到此处完成游戏
    • Boss房间，Boss掉落关键零件和大量材料，地图中最后一个Boss掉落周目关键道具，提升环境适应性，可进入更多房间
    • 挑战房间，精英敌人或某种挑战
    • 奖励房间，提供材料，回血道具，武器
    • 商店房间，消耗材料，制作回血道具，武器
    • 不可入房间，计入房间进入次数，但场景显示为探索后退回
    • 关键进展房间，用于剧情碎片收集
    • 隐藏房间
  • 不可入房间分为永久不可入房间(某周目不可入)和因环境导致的临时不可入房间
    • 海洋房间，完全是水，但在温度-1/-2时可进入
    • 岩浆白昼，极端炎热，光照为无穷大
    • 寒冰黑夜，极端寒冷，光照为0
    • 山脉高原，完全障碍地形
    • 无尽深渊，完全障碍地形
  • Boss难度按第一次进入Boss房为准，难度随已击杀boss递增

房间

尺寸

以角色为一个单位方格为基准，标准房间单位为16x9，占据横向显示设备（PC/手机），组合房间为基准房间的倍数，如Boss房长宽分别为基准房间的2倍。

门

门生成于房间的四边，一个房间最多有四个门，门在两个房间中一定是成对连通的

按顺序从(0,0)开始生成门。

按如下优先级修正门：

  • 隐藏房间只有一个门可入。

障碍物

障碍物以单元格为基准随机生成，障碍物分为：

  • 不可重叠可破环障碍物，拥有可破坏次数N，被攻击N次（次数变为0）后障碍物消除
    • 岩石容器，拥有1次可破坏次数
    • 巨型岩石，拥有3次可破坏次数
    • 冰块容器，拥有1次可破坏次数
    • 巨型冰块，拥有3次可破坏次数
  • 不可重叠不可破坏障碍物
    • 水
  • 可重叠不可破坏障碍物
    • 岩浆池，可重叠，重叠超过0.5s后造成持续伤害

障碍物转化路径

  1. 首次进入房间时按星球属性随机和房间预设生成，比如水(10%),土(90%)，如果水含量大于75%，则全部置为水
  2. 根据预设房间属性生成障碍物
    • 比如土转变为岩石容器(5%),巨型岩石(5%)
    • 水转变为冰块容器(5%),巨型冰块(5%)
  3. 根据温度调整形态
    • 3 100%转变为岩浆池
    • 2 水/巨型冰块/冰块容器先转变为土，土/巨型岩石/岩石容器转变为岩浆池(25%)
    • 1 水/冰转变为土
    • 0 不做处理
    • -1 水转变为冰面(30%)/冰容器(10%)，冰面会使所有移动物打滑，具有抗性物品的除外
    • -2 水转变为冰面(60%)/巨型冰块(30%)/冰容器(10%)
    • -3 水转变为冰面(30%)/巨型冰块(60%)/冰容器(10%)

环境

环境随时间变化

可变因子包括：

  • 温度，影响地形，能力，敌人生成
  • 光照，影响视野，敌人生成
  • 水分布，影响地形，一周目不可下水，可能转变为冰障碍物

环境温度按等级取整到：

  • 3 岩浆白昼，不可入
  • 2 极端炎热，地表生成岩浆池
  • 1 热，地表无水
  • 0 正常，地表有液态水
  • -1 冷，部分水结冰
  • -2 极端严寒，全部水结冰，生成冰障碍物
  • -3 寒冰黑夜，不可入

当前经度温度按0 - -3 - 0 - 3 - 0均匀分布
按进入房间次数，左移/右一(自转，一颗星球二选一)一个标准房间

光照强度和温度的关系：

光照强度影响视野半径，为基础视野8，为装备带来的视野加成

材料

  • 收集材料，材料可用于
    • 制作武器
    • 强化武器
    • 制作功能性物品
    • 制作消耗品
  • 材料从敌人掉落或环境开采
  • 武器强度与材料种类和数量相关，非关键零件质量与材料种类和数量相关
  • 收集关键零件和材料修复飞船
  • 材料简单分为金属M与有机物O

武器

基本属性

武器形态，先以广义的枪为原型，每把枪具有：

  • 弹夹容量Bullets，没有子弹限制，释放完弹夹容量的子弹后，再次射击会出发点充能补充
  • 特殊弹夹容量SpecialBullets，同上，初始为1
  • 最小攻击间隔时间AttackInterval
  • 充能时间RechargeTime
  • 同时发射的子弹数量ConcurrentBullets
  • 攻击距离AttackRadius，子弹按鼠标/虚拟键盘指示方向发射
  • 子弹散射角度SpreadAngle
  • 子弹偏转角度RandomErrorAngle +-5

以上属性均可强化

基本约束

  • 角色仅能携带一把武器
  • 武器会过热，需要充能
  • 武器有两种攻击模式，普通攻击和特殊攻击，分别消耗对应子弹
  • 特殊攻击冷却时间较长
  • 武器可使用材料升级
  • boss可掉落武器
  • 可能开出武器
  • 武器可分解回收一定数量的材料

子弹类型（deprecated）

  • 普通，不叠加任何伤害
  • 电磁，持续伤害(2/s)，对属性免疫无效，有几率(1%)中断行动
  • 灼烧，持续伤害，升高敌人感知温度1，可以消除温度契合伤害
  • 冰冻，持续伤害，降低敌人感知温度1，可以消除温度契合伤害
  • 爆炸，一次性额外伤害
  • 引力，有几率(5%)减速敌人N秒

可选择当前装备子弹类型，同一时间仅能使用一种普通子弹和一种特殊子弹

子弹属性

子弹随机N个属性槽，服从泊松分布

  • 触发事件持续时间 Persistance
  • 触发间隔 Interval
  • 触发几率 Probability
  • 触发行为 Event
    • 伤害
      • 元素属性 DamageType
      • 伤害量 Damage
    • 减速
      • 减速比例
      • 中断行动是减速的特例，减速比例为100%
    • 感知温度
      • 温度变化 +1/-1
  • 触发行为持续时间 Time
  • 飞行速度 Speed
  • 子弹半径 Radius

比如：

一次性伤害
Persistance = 0
Interval = 0
Probability = 1
Event = 伤害，Damage

特殊子弹

  • 引力场，爆炸点范围内敌人减速
  • 闪电网，多个闪电点间形成闪电连，给予穿过的敌人持续伤害，离开则伤害中止
  • 火焰阱，持续伤害，升高范围内温度
  • 冰冻阱，持续伤害，降低范围内温度
  • 爆炸物，爆炸点附近产生范围伤害
  • 机枪塔，爆炸点生成自动机枪塔，自动攻击附近敌人

特殊子弹可强化

  • 充能时间
  • 影响范围，半径
  • 持续时间
  • 最大并发数量
