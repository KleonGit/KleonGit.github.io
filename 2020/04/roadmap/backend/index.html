后端入门路线地图

发布于 四月 19, 2020    更新于 四月 25, 2020

一份后端入门路线图，标记了关键节点与参考资源，主要涉及后端语言以及数据库。请带着目标使用参考资源，不需要全部使用，达成目标即可。

本文主要参考developer-roadmap。

前置技能

主要是对网络和编程有初步的了解。

  • 网络与编程基础

Internet基础

了解互联网的基本原理。

主要目标：

  • 计算机之间是怎样传输信息的？
  • 什么是HTTP？
  • 什么是DNS？
  • 什么是域名？
  • 什么是服务器？
  • 什么是API？
  • 浏览器发出网页请求后到收到网页之间发生了什么？

参考资源：

  • 互联网是怎样工作的
  • HTTP教程
  • 图解HTTP
  • 计算机网络-自顶向下方法

前端基础

大致对前端的技术的三剑客有个了解即可。

  • HTML基础
  • CSS基础
  • JavaScript基础

操作系统基础

后端开发目前主流基于Linux操作系统，了解基本的Linux操作以及操作系统基本原理即可。

主要目标：

  • 计算机的主要构成是怎样的？
  • 什么是操作系统？
  • 操作系统是如何工作的？
  • 什么是内存管理、进程管理、IO管理？
  • 进程和线程的区别是什么？
  • 如何跨进程通信？什么是Remote Processor Call（RPC）？
  • POSIX基础：stdin、stdout、stderr、pipes。
  • 基本网络常识：TCP/IP协议栈，Socket，HTTP。
  • 什么是Linux？Linux有哪些版本？
  • 掌握基本的Linux命令：ls、cd、grep、find、awk、sed、curl、wget、tar、tail、head、less、more、ssh、kill、pkill、top。

参考资源：

  • Modern Operating Systems
  • Operating Systems: Three Easy Pieces
  • 操作系统之哲学原理
  • 鸟哥的私房菜-基础篇

Linux的经典中文资源，申请云服务器（现在申请云服务器很方便，学习用按量付费也很便宜，而且不用担心搞坏电脑或者电脑配置不够）或者安装Windows Linux Subsystem的看第一部分前三节和最后一节、第二部分和第三部分，本地安装（多系统或者虚拟机）的看前三部分。

  • 鸟哥的私房菜-网络篇

编程语言

挑选并学习一门你喜欢的编程语言吧！

可以先看一眼编程语言流行排行榜。

主流用作后端开发的编程语言包括：Java、Python、JavaScript、C#、C++、PHP、Go、Rust
选择自己熟悉的即可，日后切换语言也， 可触类旁通。

主要目标：

  • 什么是编译型语言？什么是解释型语言？二者有何差别？
  • 什么是强类型语言？什么是弱类型语言？二者有何差别？
  • 什么是动态类型语言？什么是静态类型语言？二者有何差别？
  • 什么是过程式编程？什么是面向对象编程？什么是函数式编程？
  • 挑选并入门一门语言。

参考资源：

  • 语言类型

代码版本管理

不多说，必学。

git是代码版本管理工具，github是代码托管平台，可搭配使用。如果需要本地自建托管服务，可以使用gitlab。

主要目标：

  • 掌握基础的git命令：config、clone、add、commit、push、pull
  • 熟悉clone github项目的方法

参考资源：

  • 手把手git教程
  • git documentation

关系型数据库

数据库是许多后端系统的核心，而关系型数据库又是经典数据库类型。

主要目标：

  • 什么是数据库？
  • 数据库有几种类型？
  • SQL是什么？
  • 学习一种关系型数据库，了解常用的概念：表、键、索引等。
  • 掌握基本操作，能直接使用SQL操作数据库。
  • 使用熟悉的编程语言操作数据库。
  • 了解ORM，使用熟悉的编程语言相关的ORM库操作数据库。

参考资源：

  • MySQL教程
  • PostgreSQL教程

非关系型数据库

非关系型数据库近来也得以广泛应用。

主要目标：

  • 什么是非关系型数据库？
  • 熟悉MongoDB的使用。

参考资源：

  • MongoDB教程

数据库原则

主要目标：

  • 什么是ACID？
  • 什么是N+1问题？

API

主要目标：

  • 什么是API？
  • 了解JSON文件格式。
  • 了解Restful API。
  • RPC与HTTP API调用有什么区别？
  • 了解gRPC。
  • 列举熟悉的编程语言的主要后端框架。
  • 挑选一个后端框架，搭建简单的API服务器。

鉴权

主要目标：

  • 鉴权有哪些方式？
  • Token、JWT、Session分别是怎样鉴权的？有什么优劣？
  • 结合数据库、API、JWT鉴权，实现用户注册与登录功能。

缓存

主要目标：

  • 为什么要有缓存？使用缓存会带来哪些问题？
  • CDN、服务端缓存、客户端缓存有何优劣？

网络安全

主要目标？

  • 什么是Hash算法？有哪些知名算法？
  • 目前哪些Hash算法有安全性问题？
  • 为什么要使用HTTPS？相比HTTP安全在哪里？
  • 什么是CORS？该如何设置？

测试

主要目标：

  • 为什么要有测试？
  • 测试分哪些层级？
  • 你熟悉的编程语言有哪些常用的测试框架？

CI/CD

主要目标：

  • 什么是持续集成、持续部署？
  • 主要步骤是哪些？
  • 使用的代码托管平台上提供的CI服务。

设计开发原则

主要目标：

  • DDD、TDD

后端架构模式

主要目标：

  • 了解后端架构演变历程。

搜索引擎

主要目标：

  • 了解搜索引擎原理。

消息中间件

主要目标：

  • 了解信息中间件的功能和原理。

虚拟化和容器化

主要目标：

  • 什么是虚拟化？
  • 为什么需要虚拟化？
  • 虚拟化有哪些方法？
  • 虚拟机和容器有什么区别？
  • 熟悉docker的使用。

GraphQL

图数据库

WebSocket

Web服务器

主要目标：

  • 什么是网络服务器？为什么需要网络服务器？
  • 了解Nginx、Apache。

可扩展式开发

主要目标：

  • 迁移策略
  • 横向扩展和纵向扩展
  • 可观测性与日志
